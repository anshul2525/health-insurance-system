# Health Insurance Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on a health insurance dataset to understand how different factors influence medical insurance charges. The analysis focuses on identifying patterns, trends, and relationships within the data using statistical methods and visualizations.

---

## Dataset Description
The dataset contains information about individuals and their health insurance details.

**Features included:**
- Age  
- Gender  
- Body Mass Index (BMI)  
- Number of children/dependents  
- Smoking status  
- Region  
- Insurance charges  

The dataset is used for educational and analytical purposes.

---

## Tools and Libraries Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Analysis Performed
- Data loading and initial inspection  
- Checking and handling missing or duplicate values  
- Univariate analysis of individual features  
- Bivariate analysis to study relationships between variables  
- Data visualization using plots and charts  
- Correlation analysis  
- Observations and insights at each stage  

---

## Key Insights
- Smoking status has a significant impact on insurance charges  
- Insurance costs generally increase with age  
- Higher BMI is associated with higher medical expenses  
- Smokers tend to have much higher insurance charges compared to non-smokers
